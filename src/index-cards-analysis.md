# index.njk 卡片分析

## 卡片結構分析

### 1. About Us 卡片 (第 86 行)
**特點**:
- 白色背景 (`bg-white`)
- 特殊的 group hover 效果
- 內含 SVG 圖標的連結
- 品牌卡片樣式 (`brand-card`)

**遷移建議**: 
- 保留現有結構，因為有特殊的互動效果
- 可以考慮提取為專用的 macro（如 `brand-card.njk`）

### 2. Philosophy 卡片 (第 100 行)

- 白色文字
- 特殊的 group hover 效果
- 內含 SVG 圖標的連結
- 品牌卡片樣式 (`brand-card`)

**遷移建議**:
- 保留現有結構，因為有特殊的樣式和互動
- 可以考慮提取為專用的 macro（如 `brand-card.njk`）

### 3. The Ritual 卡片 (第 167 行)
**特點**:

- 裝飾元素（圓形邊框、印章）
- 特殊的視覺效果

**遷移建議**:
- 保留現有結構，因為結構太複雜

**遷移建議**:
- 保留現有結構，因為有特殊的設計
- 可以考慮提取為專用的 macro（如 `booking-card.njk`）

## 總結

這些卡片都有特殊的設計和互動效果，不適合直接使用標準的 `card` macro。

**建議**:
1. 保留現有結構
2. 未來可以考慮為這些特殊卡片創建專用的 macros
3. 或者創建一個更靈活的 card macro variant 系統
