## 核心目標

將網站流量從「工具類」（2吋照片裁切、證件照產生器）轉型到「服務類」（專業形象照攝影），透過優化的用戶旅程設計、說服性文案，以及語義化內部連結策略，提升轉化率與「形象照」相關關鍵字的搜尋排名。

### 現況分析

根據 Google Search Console 數據：

- **目前流量來源**：主要來自「工具」與「DIY」需求
  - 2吋照片裁切線上（2,107 點擊）
  - 2吋大頭照產生器（928 點擊）
  - 證件照製作（487 點擊）

- **問題診斷**：
  - 搜尋意圖不符：搜尋這些關鍵字的使用者，大部分是想「免費」、「線上解決」、「自己動手做」證件照
  - 與「付費專業形象照服務」是截然不同的客群
  - 「形象照」相關字詞尚未發酵：在 Top 10 列表中，完全沒有看到「形象照」、「專業攝影」、「個人寫真」等關鍵字

## 導覽架構策略（新增「服務」導覽項目）

### 核心原則

建立「服務」導覽項目，組織主要服務頁面，讓用戶更容易找到他們需要的服務。這樣做的好處：

1. **清晰的服務導覽** - 用戶可以直接找到服務相關頁面
2. **提升 SEO 價值** - 導覽列中的服務連結有助於 SEO
3. **符合用戶搜尋意圖** - 「服務」比「作品分類」更直接
4. **組織架構更清晰** - 將服務頁面與作品展示分開

### 推薦的「服務」導覽架構（兩欄式）

**新增「服務」下拉選單，使用 megamenu 結構（與「拍攝指南」和「作品分類」一致）：**

**欄位 1：主要服務**
- 專業形象照 → `/services/portrait/` (新頁面，放在第一位，最高優先級)
- 韓式證件照 → `/blog/korean-id/` (現有)
- 畢業照服務 → `/blog/graduation/` (現有)
- 企業團體照 → 可連結到作品分類或建立專頁

**欄位 2：其他服務**
- 情侶寫真 → `/blog/couples/` (現有)
- 全家福 → `/blog/family/` (現有)
- 寵物寫真 → `/blog/pet/` (現有)
- 醫師袍形象照 → `/blog/medical/` (現有)

**導覽列結構：**
```
價目表 | 服務 ▼ | 拍攝指南 ▼ | 作品分類 ▼ | 關於我們 | [線上預約]
```

**技術實作：**
- 使用與現有 `dropdown megamenu` 相同的結構
- 使用 `megamenu-container` 和 `megamenu-column` 類別
- 確保響應式設計（行動版與桌面版）

## 用戶旅程分析與策略

### 用戶角色定義

1. **工具使用者（現有流量）**：需要快速解決證件照問題，預算敏感，DIY 傾向
2. **潛在服務客戶**：意識到形象照的重要性，但對價格、流程、效果有疑慮
3. **決策階段客戶**：已決定拍攝，正在比較選擇

### 轉化路徑設計

```
工具頁面 → 痛點喚醒 → 價值教育 → 社會證明 → 低門檻 CTA → 預約頁面
```

### 2. 建立「形象照」專屬 SEO 著陸頁（CX 優化版）

**檔案：** `src/services/portrait.njk` (新檔案，放在 services 目錄下)

**路徑：** `/services/portrait/` (更有組織，符合服務導覽架構)

**注意：** 需要建立 `src/services/` 目錄（如果不存在）

**SEO 關鍵字策略：**
- 核心關鍵字：台北專業形象照推薦、形象照、專業攝影、個人寫真
- 職業別長尾：醫師形象照、律師形象照、講師形象照、房仲形象照、求職形象照

- 地區長尾：台北形象照、信義區形象照、中山區形象照

**頁面結構（AIDA 框架 + 說服性文案）：**

**Hero Section（Attention - 注意力）**
- Title: `台北專業形象照推薦 | 個人品牌攝影、職場人像寫真 - 好時有影`
- H1: `台北專業形象照推薦 | 打造職場第一印象的攝影服務`
- 副標題（情感觸發）：`證件照可以用工具生成，但您的專業形象不能。LinkedIn、履歷表、求職網站，都需要一張真正能代表「您是誰」的專業形象照。`
- 立即 CTA：`免費諮詢拍攝方案`（低門檻，降低決策壓力）

**Section 1: 痛點喚醒（Interest - 興趣）**
- H2: `為什麼需要專業形象照？（而非一般證件照）`
- 文案策略：
  - 對比情境：「證件照 vs 形象照：放在 LinkedIn 上的差異」
  - 情感連結：「履歷表很完美，但頭像還是那張尷尬的舊照片？」
  - 職業價值：「醫師、律師、講師、企業主的第一印象，從形象照開始」
- 視覺：並排對比圖（證件照 vs 形象照在 LinkedIn 上的呈現）

**Section 2: 價值主張（Desire - 渴望）**
- H2: `適合對象：誰需要專業形象照？`
- 文案策略：使用「您是否...」問句，讓用戶自我認同
  - 「您是醫師、律師、講師、企業主，需要建立專業形象？」
  - 「您正在求職，履歷表需要一張能加分的照片？」
  - 「您的 LinkedIn 頭像還是大學畢業照？」
- 視覺：職業別作品集（醫師、律師、講師等，使用現有 portfolio）

**Section 3: 差異化說明（Desire - 深化渴望）**
- H2: `韓式證件照 vs 專業形象照：差異在哪裡？`
- 文案策略：教育型內容，建立專業權威
  - 表格對比：用途、風格、適用場景
  - 強調：「證件照解決『規格問題』，形象照解決『形象問題』」
- 視覺：作品集展示（linkedin-portrait, lawyer-portrait, doctor-coat 等）

**Section 4: 社會證明（Trust - 建立信任）**
- H2: `超過 1000+ 位專業人士的選擇`
- 文案策略：
  - 客戶見證（使用現有 testimonial 結構）
  - 企業客戶名單（Amazon、MSD、台大醫院等）
  - 數據證明：「90% 客戶在拍攝後 3 個月內更新 LinkedIn 頭像」
- 視覺：客戶見證卡片 + 企業 Logo 牆

**Section 5: 拍攝流程（Reduce Friction - 降低摩擦）**
- H2: `拍攝流程：簡單 4 步驟，無壓力體驗`
- 文案策略：強調「簡單」、「無壓力」、「專業引導」
  - 步驟視覺化（使用現有的 step-list 組件）
  - 強調：「不需要自己想姿勢，攝影師會引導您」

**Section 6: 價格透明（Reduce Friction - 消除疑慮）**
- H2: `價格透明，以張計價，不強迫購買套組`

  - 強調：「小資族友善」、「可單獨購買」
  - 連結到價目表頁面

**Final CTA Section（Action - 行動）**
- H2: `準備好，讓自己被好好看見了嗎？`
- 文案策略：情感召喚 + 低門檻行動
  - 主 CTA：`立即預約拍攝`（連結到 `/booking/`）
  - 次 CTA：`查看完整作品集`（連結到首頁 portfolio）
  - 信任元素：「免費諮詢，無壓力了解方案」

**技術實作：**
- 使用 `base-layout.njk` layout
- 參考 `src/blog/profile.njk` 的內容結構與樣式
- 使用現有的 `.module`, `.module-gray`, `.layout-split`, `.card--testimonial` 等組件

## 導覽架構優勢分析

### 為什麼建立「服務」導覽項目更好？

1. **清晰的服務組織** - 用戶可以直接找到服務相關頁面，符合搜尋意圖
2. **提升 SEO 價值** - 導覽列中的服務連結有助於 SEO，特別是形象照放在第一位
3. **組織架構更清晰** - 將服務頁面與作品展示分開，降低認知負擔
4. **多層次導覽策略** - 結合導覽列、首頁推薦、工具頁面 CTA、內部連結，全方位觸及用戶
5. **符合用戶習慣** - 「服務」比「作品分類」更直接，符合 B2B 和專業服務的用戶習慣

### 導覽策略總結（多層次導覽）

✅ **導覽列「服務」下拉選單**：主要入口，兩欄式結構，形象照放在第一欄第一位  
✅ **首頁推薦區塊**：觸及所有首頁訪客  
✅ **工具頁面 CTA**：精準轉化工具使用者  
✅ **相關文章連結**：觸及已對形象照有興趣的用戶  
✅ **價目表連結**：觸及正在考慮價格的用戶  
✅ **Footer 連結**：提供完整的服務導覽（低調但有效）
