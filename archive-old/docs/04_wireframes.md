## 4.2 Layout Outline per Page

### Page 1: 首頁 (`/`)

#### HEADER (全站共用)
- **位置**: 頁面頂部，固定定位（sticky）
- **內容**:
  - Logo（左側）
  - 導覽選單（桌面版：水平排列；行動版：漢堡選單）
    - 首頁
    - 價目表
    - 作品集（下拉選單）
    - 關於我們
    - 預約
  - 預約按鈕（右側，桌面版）

---

#### HERO SECTION
- **位置**: 首屏，Header 下方
- **佈局**: 兩欄式（桌面版）或單欄（行動版）
  - **左欄**: 文字內容
    - 副標題（Kicker）: "台北專業攝影工作室"
    - 主標題 (H1): "好時有影"
    - 副標題: "讓每一個「好時」，都有你我的「身影」"
    - 描述文字: "台北專業履歷・形象照攝影｜專為頂尖學府與百大企業打造的專業形象"
    - CTA 按鈕組:
      - 主要按鈕: "立即預約"
      - 次要按鈕: "查看價目"
  - **右欄**: Hero 圖片
    - 單張大圖（16:9 比例）
    - 支援 lazy loading（但首屏使用 eager）

---

#### PORTFOLIO GALLERY SECTION
- **位置**: Hero 下方
- **標題**: "找到你的專屬風格" (H2)
- **篩選器**:
  - 水平滾動的按鈕組（行動版）或固定排列（桌面版）
  - 按鈕類型: 韓式證件照、專業形象照、全身形象照、等（共 18 個分類）
  - 預設選中: "韓式證件照"

  - 每個作品項目:

    - 無文字標題（純視覺展示）

## 4.3 Interaction Description

### Navigation Menu

#### 桌面版
- **預設狀態**: 水平排列，所有連結可見
- **Hover 狀態**: 連結文字顏色改變，底線出現（如適用）
- **下拉選單**: 
  - 滑鼠懸停在「作品集」上時，顯示下拉選單
  - 下拉選單包含所有作品分類連結
  - 滑鼠移開時，下拉選單自動隱藏

#### 行動版
- **預設狀態**: 漢堡選單按鈕可見，導覽選單隱藏
- **點擊漢堡按鈕**: 
  - 導覽選單從側邊滑入（或從上方展開）
  - 漢堡按鈕圖示變為 X
  - 背景可能變暗（overlay）
- **點擊外部區域**: 導覽選單自動關閉
- **點擊 Esc 鍵**: 導覽選單關閉
- **點擊連結**: 導覽選單自動關閉（行動版）

---

### Portfolio Filter

- **點擊篩選按鈕**:
  - 按鈕狀態變為 active（視覺突出）
  - 其他按鈕變為 inactive
  - 作品網格立即更新，只顯示該分類的作品
  - 其他分類的作品隱藏（使用 `hidden` 屬性）
  - URL hash 更新為 `#filter=category-name`（支援分享和書籤）

- **鍵盤導覽**:
  - Tab 鍵可在篩選按鈕間移動
  - Enter 或 Space 鍵可觸發篩選

---

### FAQ Accordion

- **點擊問題按鈕**:
  - 答案內容展開（滑動動畫）
  - 按鈕圖示改變（例如：+ 變為 -）
  - 其他問題的答案自動收合（如為單一展開模式）
  - 或保持其他問題的展開狀態（如為多重展開模式）

- **鍵盤導覽**:
  - Tab 鍵可在問題間移動
  - Enter 或 Space 鍵可展開/收合答案

---

### Scroll Behavior

- **平滑滾動**: 點擊錨點連結時，頁面平滑滾動到目標位置
- **Sticky Header**: Header 在頁面滾動時保持固定在頂部
- **Scroll Animations**: 
  - 某些區塊在滾動進入視窗時觸發淡入動畫
  - 使用 Intersection Observer API
  - 尊重使用者的 `prefers-reduced-motion` 設定

---

### Image Loading

- **Lazy Loading**: 
  - 所有非首屏圖片使用 `loading="lazy"` 屬性
  - 圖片在進入視窗時才開始載入
- **Hero Image**: 
  - 使用 `loading="eager"` 優先載入
- **載入失敗處理**: 
  - 如果圖片載入失敗，顯示預設佔位圖或錯誤訊息

---

### Form Interactions (如適用)

- **預約表單**:
  - 輸入欄位有清晰的標籤和提示
  - 即時驗證（如適用）
  - 提交按鈕有載入狀態
  - 成功/錯誤訊息明確顯示

---

### External Booking System

- **Iframe 嵌入**:
  - 如果使用 iframe 嵌入外部預約系統，顯示載入狀態
  - 如果載入失敗，顯示錯誤訊息和替代聯繫方式
- **直接連結**:
  - 點擊預約按鈕後，在新分頁或當前分頁開啟外部預約系統
  - 提供「返回網站」連結（如適用）
