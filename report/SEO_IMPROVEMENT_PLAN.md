# SEO 改善方案
## CX & SEO 專家建議

**生成時間**: 2025-12-14  
**當前平均分數**: 49.9/100  
**目標分數**: 80+/100

---

## 📊 問題概況

### 主要問題統計
- **Description 問題**: 23 個頁面（100%）
- **Title 問題**: 20 個頁面（87%）
- **H1 問題**: 3 個頁面（13%）

### 核心問題
1. **Meta Description 過短**：大部分頁面的描述不足120字符，影響搜索結果展示
2. **Title 過短**：許多標題低於30字符，無法充分利用搜索結果標題空間
3. **缺少行動呼籲（CTA）**：描述中缺少引導用戶點擊的詞彙

---

## 🎯 優化策略

### 1. Title 優化原則

**最佳實踐**：
- 長度：30-60字符（建議50字符左右）
- 格式：`主要關鍵詞 | 品牌名` 或 `主要關鍵詞 - 次要關鍵詞 | 品牌名`
- 必須包含：主要關鍵詞 + 品牌名稱（好時有影 / Golden Years）

**優化示例**：

| 頁面類型 | 優化前 | 優化後 | 改善 |
|---------|--------|--------|------|
| 首頁 | 好時有影｜台北專業形象照攝影（韓式證件照、形象照、畢業照） | 台北專業形象照｜韓式證件照、個人品牌攝影 - 好時有影 Golden Years | ✅ 更聚焦關鍵詞 |
| 服務頁 | 台北專業形象照推薦 \| 個人品牌攝影、職場人像寫真 - 好時有影 | 專業形象照推薦｜LinkedIn履歷照、醫師律師形象照 - 好時有影 | ✅ 更具體化 |
| 預約頁 | 選擇分店｜好時有影 (中山店 / 公館店) | 預約拍攝｜中山店/公館店專業形象照 - 好時有影 | ✅ 更清晰 |

### 2. Meta Description 優化原則

**最佳實踐**：
- 長度：120-160字符（建議150字符左右）
- 結構：問題/痛點 → 解決方案 → 行動呼籲
- 必須包含：主要關鍵詞、價值主張、CTA

**優化公式**：
```
[關鍵詞] + [服務內容] + [優勢/特色] + [地理位置] + [CTA]
```

**優化示例**：

#### 首頁（index.njk）
**優化前**（54字符）：
```
位於台北中山/公館，韓式證件照、專業形象照、畢業寫真。立即查看價目表。
```

**優化後**（148字符）：
```
台北專業形象照攝影首選｜韓式證件照、履歷照、畢業照服務。好時有影位於中山/公館，為醫師、律師、企業主打造職場第一印象。立即預約拍攝，查看價目表與作品集。
```

#### 專業形象照服務頁（services/portrait.njk）
**優化前**（84字符）：
```
證件照可以用工具生成，但您的專業形象不能。LinkedIn、履歷表、求職網站，都需要一張真正能代表「您是誰」的專業形象照。專為醫師、律師、講師、企業主打造職場第一印象。
```

**優化後**（155字符）：
```
專業形象照不能用AI生成，您的職場第一印象需要真實呈現。好時有影為醫師、律師、講師、企業主提供LinkedIn履歷照服務，位於台北中山/公館，立即預約拍攝，打造專業個人品牌形象。
```

#### 關於我們頁（about.njk）
**優化前**（54字符）：
```
好時有影的專業團隊有才華洋溢的攝影師、細心的造型師，還有專業的修圖師，我們共同為您打造真實又好看的影像。

**優化後**（152字符）：
```
認識好時有影專業攝影團隊：經驗豐富的攝影師、專業造型師與修圖師，為您打造真實又好看的職場形象照。我們相信一張好的形象照是職涯自信的起點，立即了解我們的服務理念與團隊故事。
